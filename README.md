Basic-Projectject是一个简易的Android基础项目，方便您快速进行开发。
包含以下内容：
- BaseActivity、BaseFragment
- Activity栈管理
- 异常信息收集
- 日志打印
- 丰富的工具类

##使用步骤
第一步：
```gradle
repositories {
    maven {
        url 'https://dl.bintray.com/qq8585083/maven/'
    }
}
....
dependencies {
    compile 'com.hy.basicproject:basicproject:1.0.0'
}
```
